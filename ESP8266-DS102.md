# ESP8266-DS102

## Settings
```
Backlog0 Timezone 99; TimeStd 0,0,10,1,3,60; TimeDst 0,0,3,1,2,120
Backlog DeviceName KMMR1; FriendlyName1 KMMR1; 
Backlog ButtonTopic 0;  SetOption73 1; SetOption32 20;
```

## Rules
- Position (0-100%) 100 open
- Mem1 = 0=open, 1=close, 2=stop
- Mem1 = Last Command (open, close)
```
Rule1
  ON Button1#state=10 DO Publish shellies/shellydimmer2-4C75253381CE/light/0/set {"turn": "toggle"} ENDON
  ON Button1#state=11 DO Publish shellies/shellydimmer2-4C75253381CE/light/0/set { "brightness": 25, "turn": "on" } ENDON
  ON Button1#state=3 DO Publish shellies/shellydimmer2-4C75253381CE/light/0/set { "brightness": 100, "turn": "on" } ENDON
  ON mqtt#connected DO Subscribe HzOn, shellies/HZ_DG/status/switch:0, output ENDON
  ON Event#HzOn=true DO LedPower1 1 ENDON
  ON Event#HzOn=false DO LedPower1 0 ENDON
  ON Button1#state=13 DO Publish shellies/HZ_DG/rpc { "method":"Switch.Toggle","params": { "id":0 }} ENDON
```

```
Rule2
  ON mqtt#connected DO Subscribe Pos1, shellies/shellyswitch25-C45BBE6B07DC/roller/0/pos ENDON
  ON Event#Pos1=0 DO mem1 0 ENDON
  ON Event#Pos1=100 DO mem1 1 ENDON
  ON Button2#state DO event ROLLER1=%mem1% ENDON

  ON event#ROLLER1==0 DO Backlog Publish shellies/shellyswitch25-C45BBE6B07DC/roller/0/command open; mem1 2; mem2 1 ENDON

  ON event#ROLLER1==2 DO Backlog Publish shellies/shellyswitch25-C45BBE6B07DC/roller/0/command stop; mem1 %mem2% ENDON
  ON mqtt#connected DO Subscribe Pos2, shellies/shellyswitch25-C45BBE5FB259/roller/0/pos ENDON
  ON Event#Pos2=0 DO mem3 0 ENDON
  ON Event#Pos2=100 DO mem3 1 ENDON

  ON event#ROLLER2="" DO Backlog Publish shellies/shellyswitch25-C45BBE5FB259/roller/0/command open; mem3 2; mem4 1 ENDON
  ON event#ROLLER2==0 DO Backlog Publish shellies/shellyswitch25-C45BBE5FB259/roller/0/command open; mem3 2; mem4 1 ENDON
  ON event#ROLLER2==1 DO Backlog Publish shellies/shellyswitch25-C45BBE5FB259/roller/0/command close; mem3 2; mem4 0 ENDON
  ON event#ROLLER2==2 DO Backlog Publish shellies/shellyswitch25-C45BBE5FB259/roller/0/command stop; mem3 %mem4% ENDON
```
